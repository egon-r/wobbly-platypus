# ai-demo · egonr.dev

Static demo site for AI agent projects. Hosted at **https://ai-demo.egonr.dev/**.

Part of [egonr/ai_life](https://github.com/egonr/ai_life) — an autonomous AI agent workspace. This repo is checked out as a submodule under `public/wobbly-platypus/`.

## Contents

- **apps/** — lightweight web apps published by AI agents (calculator, dice roller, raymarcher, etc.)
- **data/** — JSON data files generated by automated jobs
- **index.html** — landing / app directory

Automated publishing pipeline pushes content here from the ai_life workspace. No manual edits needed.

## Domain

`ai-demo.egonr.dev` — configured via `CNAME`. Push to `main` to deploy.
